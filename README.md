# 📚 English Dictionary Web App

A modern and responsive English dictionary web application built using Tailwind CSS. This project allows users to search for word meanings in a clean and user-friendly interface.

---

## 📖 Project Overview

This dictionary web app helps users quickly find meanings of English words. It is designed with a simple and clean UI to ensure a smooth and distraction-free user experience.

## ⚙️ Installation & Setup Guide

Follow these steps to run the project locally:

```bash

git clone https://github.com/your-username/your-repo-name.git

# Go to project folder
cd your-repo-name

# Install dependencies
npm install

# Run the project
npm run dev
```
